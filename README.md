# Mi Tienda Online

## Descripción

Proyecto de e-commerce desarrollado con HTML, CSS y JavaScript.

La página consume productos desde una API REST mediante Fetch,
permite agregar productos a un carrito dinámico y guarda la información
utilizando LocalStorage.

## Funcionalidades

- Catálogo dinámico de productos.
- Carrito de compras.
- Edición de cantidades.
- Eliminación de productos.
- Persistencia del carrito.
- Formulario de contacto con Formspree.
- Diseño responsive.
- Optimización básica SEO y accesibilidad.

## Tecnologías utilizadas

- HTML5
- CSS3
- Flexbox
- Grid
- JavaScript
- API REST
- LocalStorage

## Instalación

Clonar el repositorio y abrir index.html en el navegador.
